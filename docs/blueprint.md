# **App Name**: Media Uploader

## Core Features:

- Media Upload: Allow users to upload photos and videos.
- Firestore Storage Integration: Save uploaded media files to Firestore Storage.
- Progress Tracking: Show upload progress to the user.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), reflecting stability and trust.
- Background color: Light gray (#F0F0F0), provides a neutral backdrop.
- Accent color: Teal (#009688), offers a touch of vibrancy and modernity.
- Font: 'Inter', sans-serif, suitable for body and headlines.
- Use clear and intuitive icons for upload, save, and progress indicators.
- Simple, clean layout with a focus on ease of use. Drag-and-drop functionality for uploads.
- Subtle animations to indicate loading and completion states.
